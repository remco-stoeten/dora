# Dora docs

Dora connects to local files, hosted databases, and team infrastructure from one desktop SQL client. These docs cover setup, provider connection guides, and lower-level developer references.

Dora connects to any PostgreSQL, MySQL, or libSQL host the same way. Railway, Render, Vercel Postgres, Fly.io, Aiven, DigitalOcean, Crunchy Bridge, Timescale, AWS RDS, Azure, Google Cloud SQL, CockroachDB Cloud, TiDB Cloud, PlanetScale, and self-hosted databases all work with a standard connection string, recognized automatically with the right engine and SSL.
